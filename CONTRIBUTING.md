# Contributing

Thanks for helping out! **PRs and issues are welcome.**

- Found a bug, a typo, or something confusing? [Open an issue](../../issues).
- Have a fix or an improvement? Send a pull request.

A few small asks:

- Keep each episode self-contained (its own `README.md`, `src/`, `tests/`, and `Makefile`).
- Run `make test` in the episode you changed before opening a PR.
- If you've installed [pre-commit](https://pre-commit.com/), run `pre-commit run --all-files`
  so formatting and lint stay consistent.

That's it. Be kind, keep it beginner-friendly, and have fun.

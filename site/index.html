LevelUp Dev Lab

Help entry-level devs think and build like engineers.

Practical episodes on automation, best practices, dev tooling, and building your own tools — in Python, Java, and JavaScript. Every episode ships as a runnable project you can clone, run, and break.

Four pillars

  • Automation Recipes

    Small scripts that delete busywork. Each recipe solves a chore you would otherwise do by hand.

  • Best Practices

    The habits that separate juniors from engineers: naming, tests, error handling, observability.

  • Dev Tooling

    Configure and master the tools you already use — editors, linters, formatters, CI.

  • Build Your Own Tooling

    Go from tool user to tool maker. CLIs, scaffolders, and helpers built from the standard library up.

Six-month roadmap

  1. Jul 2026 Automation foundations — folder reports, file chores, and the anatomy of a small tool.
  2. Aug 2026 Clean code fundamentals — naming, functions, and the same ideas across Python, Java, and JavaScript.
  3. Sep 2026 Dev tooling — editor setup, formatters, pre-commit hooks, and a CI pipeline from scratch.
  4. Oct 2026 Observability basics — logging, timing, and reading what your program tells you.
  5. Nov 2026 Build your own CLI — argument parsing, exit codes, and testable command-line tools in Python and Node.
  6. Dec 2026 Community month — viewer contributions, code review episodes, and the season recap.

About the creator

LevelUp Dev Lab is made by a working developer who spent the first years of the job learning things the slow way — and would rather you did not have to. No shortcuts sold here: each episode is a small, honest project with tests, a written walkthrough, and the reasoning behind every decision.

The channel favors the standard library over frameworks, plain text over slides, and working code over talk. If an episode cannot be cloned and run, it does not ship.
